# Card Logo Forge

A browser tool for the **Bless Thy Pulls** MTG series. Type a card name, pick a
metal style, and download a transparent PNG (for video overlays) or SVG (for
merch/print). Runs entirely in the browser — no build step, no dependencies.

## Styles
- **Thrash** — chrome lettering
- **Brand White** — white + heavy outline (BTP look)
- **Gothic** — heavy blackletter
- **Death** — drippy blood-red horror
- **Druid** — carved green & bronze

## Use it
Live: https://USERNAME.github.io/card-logo-forge/

Or open `index.html` locally in any browser.

## Notes
Uses real web fonts (Google Fonts), so spelling is always exactly what you type.
For illustrated looks (vines, wings, skulls woven into letters), use an AI image
generator instead — fonts can't draw those.
